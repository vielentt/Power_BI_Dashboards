# Дашборд "Data Jobs 2.0" с Power BI

![Data Jobs Dashboard 2.0](/images/poject_2_p_1.PNG)

## Введение

Ориентироваться на рынке труда в сфере обработки данных может быть непросто, поскольку информация разбросана повсюду. Этот дашборд создан для тех, кто хочет найти или сменить работу. ​​Используя реальный набор данных о вакансиях в области анализа данных по миру за 2024 год (с подробной информацией о названиях должностей, зарплатах и ​​местоположении) этот проект предлагает упрощенный одностраничный интерфейс для быстрого изучения важных рыночных тенденций и анализа уровня заработной платы.


## Навыки, использованные в проекте

При создании дашборда были использованы разнообразные ключевые навыки Power BI. Ниже представлены основные из них:
 - **Дизайн дашбордов:** Создание интуитивно понятного и визуально привлекательного макета отчета.
 - **ETL Power Query:** Выполнение очистки, обработки и преобразования данных
 - **Основные диаграммы:** Использование столбчатых, точечных, линейчатых диаграмм и временного графика для сравнения количества вакансий и отслеживания тенденций во времени.
 - **Моделирование данных:** Построение эффективных моделей данных с использованием связей (Star Schema)
 - **Основы DAX:** Создание вычислений и агрегаций для получения ключевых выводов.
 - **Использование основных визуализаций:** Столбчатые, линейные диаграммы для сравнений и анализа тенденций. Карточки для выделения ключевых показателей эффективности.
 - **Интерактивные функции:** Срезы для обеспечения динамической фильтрации данных, управляемой пользователем. Кнопки для упрощения навигации и управления представлениями отчетов (включая детализацию).


## Обзор проекта

Во второй версии проекта дашборд объединён в **единую страницу**, призванную предоставить соискателям работы самую важную информацию о рынке труда в режиме реального времени.

![Data Jobs Dashboard 2.0](/images/poject_2_p_2.PNG)

На этой странице представлена краткая информация о рынке труда в сфере обработки данных. Здесь представлены ключевые показатели эффективности (KPI), такие как **количество вакансий, требуемые навыки, медианная годовая и медианная почасовая ставки**. Вы также можете быстро увидеть **популярность навыков** (в процентах или по количеству вакансий) и сравнить **заработные платы по различным должностям**.

## Заключение

Этот обновлённый дашборд (версия 2.0) демонстрирует возможности Power BI по преобразованию обширных данных о вакансиях в мощный и оптимизированный инструмент для анализа карьерных перспектив. Она расширяет возможности **соискателей работы**, позволяя им эффективно фильтровать и изучать важные рыночные данные на одной странице, помогая им принимать обоснованные решения о следующем шаге в своей карьере.
