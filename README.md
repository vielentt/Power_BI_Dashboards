# Дашборд Data Вакансий с Power BI

![Дашборд стр. 1](/images/poject_1_p_1.PNG)

## Введение

Этот дашборд создан для **тех, кто ищет работу**, чтобы решить распространенную проблему: информация о рынке труда в сфере обработки данных разрознена и сложна для понимания. Используя реальный набор данных о вакансиях в области анализа данных по всему миру за 2024 год (включая названия должностей, зарплаты и местоположение), этот проект предоставляет единый, простой в использовании интерфейс для изучения рыночных тенденций и уровня заработной платы.

## Навыки, используемые в проекте

При создании дашборда были использованы разнообразные ключевые навыки Power BI. Ниже представлены основные из них:
 - **Преобразование данных (ETL) с помощью Power Query:** Очистка, преобразование и подготовка исходных данных для анализа путем обработки пустых полей, изменения типов данных и создания новых столбцов.
 - **Основные диаграммы:** Использование столбчатых, точечных, линейчатых диаграмм и временного графика для сравнения количества вакансий и отслеживания тенденций во времени.
 - **Геопространственный анализ:** Использование картографических диаграмм для визуализации глобального распределения вакансий.
 - **Дизайн панели управления:** Разработан интуитивно понятный и визуально привлекательный макет, с использованием как распространенных, так и нестандартных типов диаграмм для наилучшего представления данных.
 - **Интерактивные вставки:** Срезы для динамической фильтрации отчета по профессии, кнопки для удобной навигации, детализация данных для перехода от общего обзора к контекстному, подробному представлению.
 - **Карточки и таблицы:** Использование карточек для отображения ключевых показателей и таблиц для предоставления детализированных, сортируемых данных.
